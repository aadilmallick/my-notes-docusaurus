## Managing desktops

### Creating desktops

1. Click the **Task View** button on the taskbar or press **Windows + Tab** to open Task View.
2. Click **New desktop** (plus sign) to create a new virtual desktop.
3. If the Task View button is hidden, right-click the taskbar, select **Taskbar settings**, and toggle **Task view** on.

### Switching between desktops

There are multiple ways to switch in between desktops, and here they are, ranked in order by preference.

- **baseline**: Press **Windows + Tab** to open up the desktop switch view
- **decent**: Press **Windows + Ctrl + →** or **Windows + Ctrl + ←** to switch desktops.
- **best**: Use 4 fingers swipe to right or left to switch windows

## Finger swipes

- 4 fingers swiping changes between desktops
- 3 fingers swiping changes between windows
- 2 fingers swiping goes forward or backwards in browser history.
