## Glassmorphism

```css
.glassmorphism {
  background-color: transparent;
  backdrop-filter: blur(20px);
}

.glassmorphism-white {
  background-color: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(20px);
}

.glassmorphism-card {
  padding: 2rem;
  border-radius: 1rem;
}
```
